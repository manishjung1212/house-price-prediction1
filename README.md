This project predicts house prices using machine learning. It leverages Python, pandas, scikit-learn, and XGBoost to train a model and evaluate its performance.

The model can help estimate property values based on features like area, number of bedrooms, bathrooms, and more.

Features
Handles data preprocessing and missing values.
Splits data into training and testing sets.
Uses XGBoost Regressor for prediction.
Evaluates model performance using:
Mean Absolute Error (MAE)
R² Score
